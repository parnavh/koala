# 🐨 Koala

A utility discord bot

## 🏗 Development

```
bun install
bun dev
```

## 💻 Production

```
bun install --production
bun build
bun start
```

## 🐋 Docker

To start the application:

```
docker-compose up -d
```

To shut down your application:

```
docker-compose down
```

To view your application's logs:

```
docker-compose logs
```
