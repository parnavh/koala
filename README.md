# 🐨 Koala

A utility discord bot

## 🏗 Development

```
pnpm install
pnpm dev
```

## 💻 Production

```
pnpm install
pnpm build
pnpm start
```

## 🐋 Docker

To start the application:

```
docker-compose up -d
```

To shut down your application:

```
docker-compose down
```

To view your application's logs:

```
docker-compose logs
```
